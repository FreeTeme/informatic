Векторный графический редактор инженерной и деловой графики. Подходит для замены иностранных решений

О ПРОДУКТЕ

Универсальное применение

Может быть использован в организациях любого вида деятельности, в т.ч. на объектах КИИ (критическая информационная инфраструктура).

Все данные на схеме

Позволяет отображать данные из БД на схемах, настраивать связи объектов схем с объектами базы данных и другими схемами, выполнять расчёты и отображать их результаты

Включён в реестр

№ 3132 от 14.03.2017 в Едином реестре российских программ для электронных вычислительных машин и баз данных

Совместная работа

Безопасность

Контроль целостности файлов, логирование событий, автосохранение документа и другое

Широкая поддержка ОС

ФУНКЦИОНАЛЬНЫЕ ВОЗМОЖНОСТИ

Полнофункциональный редактор для создания и редактирования различных типов схем и чертежей. Включает в себя:

  • Создание технических схем и диаграмм
  • Редактирование векторной графики
  • Работа с многослойными проектами
  • Точное позиционирование элементов
  • Создание сложных композиций

Создание собственных библиотек элементов и шаблонов для повторного использования. Возможности:

  • Создание пользовательских библиотек символов
  • Сохранение и переиспользование шаблонов
  • Импорт готовых библиотек
  • Создание корпоративных стандартов
  • Управление версиями библиотек

Импорт схем и библиотек из различных форматов для работы с существующими проектами. Поддерживаемые форматы:

  • Импорт схем: Visio (vsdx, vsd, vsdm, vstx, vdw)
  • AutoCAD (dwg, dxf, dwt)
  • LibreOffice Draw (odg, fodg, otg)
  • MapInfo (mif), SVG, SVGZ
  • Растровые изображения (png, jpg, jpeg, bmp, gif, ico, tif, tiff)
  • Импорт библиотек Visio (vssx, vssm, vss), AutoCAD (dwg)

Экспорт в различные форматы файлов для совместимости с другими системами. Поддерживаемые форматы:

  • PDF, PNG, JPG для презентаций
  • SVG для веб-приложений
  • DXF, DWG для CAD-систем
  • VSDX для Microsoft Visio
  • HTML для интерактивных схем

Поддержка контейнеров и табличных структур для организации данных на схеме. Функции:

  • Создание и редактирование таблиц
  • Группировка элементов в контейнеры
  • Связанные данные и формулы
  • Сортировка и фильтрация
  • Автоматическое обновление связей

Гибкая печать на листы любых форматов, создание наборов листов, поля обрезки. Возможности:

  • Печать на листы A4, A3, A2, A1, A0
  • Настройка полей и масштабирования
  • Предварительный просмотр
  • Пакетная печать документов
  • Экспорт в PDF для печати

Быстрый переход к объектам, фильтры и поиск по атрибутам элементов и данным. Инструменты:

  • Быстрый поиск по названиям и атрибутам
  • Фильтрация по типам элементов
  • Навигация по связям между объектами
  • Закладки и избранное
  • История просмотра

Контроль целостности, разграничение доступа, журналирование и аудит операций. Функции безопасности:

  • Система ролей и прав доступа
  • Шифрование данных
  • Журналирование всех операций
  • Контроль версий и целостности
  • Резервное копирование

Создание единой цифровой модели предприятия с полной базой данных объектов и их связей. Возможности:

  • Единая база данных объектов предприятия
  • Связи между объектами и схемами
  • Иерархическая структура данных
  • Интеграция с существующими системами
  • Масштабируемая архитектура

Гибкая система управления правами доступа с ролевой моделью. Возможности:

  • Создание ролей и групп пользователей
  • Настройка прав доступа к схемам и данным
  • Профили пользователей с индивидуальными настройками
  • Контроль доступа к функциям системы
  • Аудит действий пользователей

Совместная работа нескольких пользователей над схемами в реальном времени. Функции:

  • Совместное редактирование схем
  • Система блокировок элементов
  • Разрешение конфликтов редактирования
  • Уведомления об изменениях
  • История совместной работы

Интеграция элементов схем с базой данных для создания единой информационной модели. Возможности:

  • Связь элементов схем с записями БД
  • Двусторонняя навигация между схемой и данными
  • Автоматическое обновление связей
  • Синхронизация изменений
  • Валидация целостности данных

Визуализация данных и показателей непосредственно на элементах схем. Функции:

  • Отображение атрибутов на элементах
  • Динамические показатели и метрики
  • Цветовое кодирование по значениям
  • Интерактивные подсказки с данными
  • Настраиваемые форматы отображения

Система навигации между схемами и связанными данными. Возможности:

  • Ссылки между схемами
  • Переходы к связанным данным
  • Навигация по документам
  • Контекстные меню переходов
  • История навигации

Автоматическое формирование и анализ топологии соединений. Функции:

  • Автоматическое определение трасс соединений
  • Генерация отчётов по топологии
  • Проверка корректности соединений
  • Визуализация путей соединений
  • Анализ критических путей

Полный контроль версий и процессов согласования схем. Возможности:

  • Версионирование схем и документов
  • Сравнение версий и отслеживание изменений
  • Маршруты согласования с уведомлениями
  • Аудит всех операций пользователей
  • Восстановление предыдущих версий

Инструменты разработки и автоматизации для расширения функциональности. Компоненты:

  • API для интеграции с внешними системами
  • Редактор библиотек элементов
  • Инструменты автоматизации процессов
  • Система расширений и плагинов
  • Средства разработки пользовательских решений
